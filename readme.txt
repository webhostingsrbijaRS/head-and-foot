Add custom scripts to the header and footer of your WordPress site.

== Description ==
Allows users to insert custom scripts in the header and footer via the Settings > Head & Foot Scripts menu.

== Installation ==
1. Upload `head-and-foot-scripts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Head & Foot Scripts to add your custom scripts.
